# Learning Network Skill（知识网络版）

基于掌握学习、间隔重复、主动回忆、苏格拉底提问的个人学习助手。支持多主题知识网络、兴趣挖掘（三层梯度）、复习路由、错题遗漏追踪与自动归档。

## 功能特性

- **学习模式**：新建或继续主题，生成课程与检查站，自动判定掌握程度
- **复习模式**：基于间隔重复算法，到期自动提醒，支持错题优先复习
- **恢复模式**：暂停主题复归条件检查，满足条件即可恢复学习
- **全局复习**："今天该复习什么" 一键查看所有到期复习项
- **微复习**：5 分钟快速主动回忆，微复习命中计数自动触发升格检查
- **课程规划**：跨会话持久化学习路线，新会话不丢失方向
- **知识点笔记**：自动触发创建（"XX是什么"→提议建笔记），核心洞见+关键词标签+双向链接
- **知识网络**：跨主题关联索引（瘦身）、各主题内部 `关联.md` 详细桥接、待验证假设追踪
- **兴趣挖掘（三层梯度）**：种子即时捕获 → 探索主题轻量孵化 → 正式开课+编织关联
- **兴趣挖掘归档**：`_兴趣挖掘日志/` 按月归档历史记录，支持种子溯源
- **错题追踪**：活跃遗漏汇总、跨课引用、微复习命中计数、自动归档
- **探索中主题**：轻量结构支持深度兴趣的"低摩擦探索"，含用户意图倾向字段
- **主题结构演化**：支持主题合并（重叠 > 50%）、主题拆分（课程 > 15 课可拆分）
- **主题暂停回退**：暂停原因与复归条件显式记录，防止僵尸主题堆积
- **难度波动检测**：波动较大时主动抽查前置概念，防止知识断层

## 技能文件结构

技能源码采用多文件拆分，主文件 `SKILL.md` 只做入口与路由，详细内容下沉到子文件夹：

```
learning-network-skill/
├── SKILL.md                     # 入口（Core Rule + 路由表）
├── main.md                      # 同 SKILL.md 的别名
├── _全局结构/
│   ├── 全局目录规范.md
│   ├── 全局索引模板.md
│   ├── 知识关联图谱模板.md
│   └── 兴趣挖掘日志规范.md
├── _主题内部/
│   ├── 主题结构说明.md
│   ├── 进度模板.md              # 含探索中主题简化版
│   ├── 复习计划模板.md
│   ├── 关联模板.md              # 含假设验证记录
│   ├── 错题模板.md
│   └── 活跃遗漏模板.md          # 含微复习命中列
├── _模式/
│   ├── 入口路由.md              # 七种入口优先级判断
│   ├── 学习模式.md
│   ├── 复习模式.md
│   ├── 全局复习模式.md
│   ├── 微复习模式.md
│   ├── 恢复模式.md
│   └── 兴趣挖掘与主题演化.md    # 三层梯度+合并拆分+回退
└── _规则/
    ├── 掌握程度判定.md
    ├── 难度自评与节奏.md
    ├── 复习间隔与退化规则.md
    ├── 更新规则清单.md          # 8条完整更新动作
    └── 反模式.md
```

## 安装

```bash
git clone https://github.com/Altmight/learning-network-skill.git ~/.codex/skills/learning-network-skill
```

或使用 skill-installer：

```
/install-skill https://github.com/Altmight/learning-network-skill
```

## 数据结构（用户学习数据）

所有学习数据存放在 `my-learning-log/` 目录下：

```
my-learning-log/
├── _全局索引.md              # 所有主题的导航、进度摘要（含探索中主题）
├── _知识关联图谱.md          # 跨主题索引（仅摘要，详细关联下沉到各主题 关联.md）
├── _兴趣挖掘日志/            # 历史兴趣挖掘记录按月归档
├── [主题A]/
│   ├── 进度.md               # 学习进度与状态
│   ├── 课程规划.md           # 未来课程大纲（标题+目标，非正文）
│   ├── 复习计划.md           # 间隔复习队列与记录
│   ├── 关联.md               # 本主题与其他主题的详细关联（含假设验证记录）
│   ├── 课程/                 # 每节课的讲解与检查站
│   ├── 笔记/                 # 知识点深度笔记（核心洞见+关键词+双向链接）
│   ├── 错题/                 # 错题与遗漏追踪（含微复习命中计数）
│   └── 归档/                 # 旧复习记录归档
└── ...
```
