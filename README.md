# Learning Network Skill（知识网络版）

基于掌握学习、间隔重复、主动回忆、苏格拉底提问的个人学习助手。支持多主题知识网络、兴趣挖掘、复习路由、错题遗漏追踪与自动归档。

## 功能特性

- **学习模式**：新建或继续主题，生成课程与检查站，自动判定掌握程度
- **复习模式**：基于间隔重复算法，到期自动提醒，支持错题优先复习
- **全局复习**："今天该复习什么" 一键查看所有到期复习项
- **微复习**：5 分钟快速主动回忆，适合碎片时间
- **知识网络**：跨主题关联、兴趣挖掘、意向主题池
- **错题追踪**：活跃遗漏汇总、跨课引用、自动归档

## 安装

```bash
# 克隆仓库到 Codex skills 目录
git clone https://github.com/Altmight/learning-network-skill.git ~/.codex/skills/learning-network-skill
```

或使用 skill-installer：

```
/install-skill https://github.com/Altmight/learning-network-skill
```

## 使用方式

| 用户表达 | 模式 | 说明 |
|----------|------|------|
| 我想学 X / 学习 X / 教我 X | 学习模式 | 新建或继续主题，生成课程与检查站 |
| 我想复习 X / 复习 X | 复习模式 | 打开对应主题，读取遗漏和复习计划 |
| 今天该复习什么 | 全局复习 | 读取全局索引，输出到期/过期项 |
| 看看我的错题 | 错题模式 | 读取所有主题的活跃遗漏 |
| 我多久没学过 X 了 | 间隔查询 | 返回距上次学习/复习的天数 |
| 快速过一遍 X | 微复习 | 抽取遗漏点做 5 分钟快问快答 |

## 数据结构

所有学习数据存放在 `my-learning-log/` 目录下：

```
my-learning-log/
├── _全局索引.md              # 所有主题的导航、进度摘要
├── _知识关联图谱.md          # 跨主题双向链接与兴趣池
├── [主题A]/
│   ├── 进度.md               # 学习进度与状态
│   ├── 复习计划.md           # 间隔复习队列与记录
│   ├── 课程/                 # 每节课的讲解与检查站
│   ├── 错题/                 # 错题与遗漏追踪
│   └── 归档/                 # 旧复习记录归档
└── ...
```

## 底层学习原则

1. **费曼技巧**：用最简单的语言解释复杂概念
2. **苏格拉底式提问**：不直接给答案，用追问引导推导
3. **脚手架原则**：从已知出发，逐步搭建到新概念
4. **间隔重复**：按复习计划在特定日期主动回忆
5. **知识网络生长**：每次学习都可能触发新兴趣点
