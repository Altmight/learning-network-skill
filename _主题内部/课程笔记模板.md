# `课程/NN_笔记.md` 模板

每门课可以有一个对应的知识点笔记文件，放在 `课程/` 目录下，与课程文件并列。用于记录**不是错题、不是跨主题兴趣、但值得深度记住的知识点**——洞见、关键公式、思维模型、"aha moment"。

文件名：`NN_笔记.md`（如 `08_笔记.md` 对应 `08_状态空间模型与动态系统入门.md`）。

模板：

```markdown
# 课程/XX_标题 知识点笔记

## [知识点A]
- **核心洞见**：（用一句话说清本质）
- **为什么重要**：（这个知识点改变了你对什么的理解？）
- **类比/直觉**：（不用公式，怎么跟一个外行解释？）
- **关键公式/定义**：（如果有）

## [知识点B]
- **核心洞见**：
- **为什么重要**：
- **类比/直觉**：
```
```

> **使用规则**：
> - 这是一个**可选文件**——不是每门课都需要。只有遇到需要深度记录的知识点时才创建。
> - 知识点和错题的区别：错题是你**理解错了**的东西，知识点是你**理解了但想记住深度**的东西。
> - 每学完一门课，助手可以简短问一句："这节课有没有哪个概念你想单独记一下？"
> - 复习时如果相关知识点被触及，助手可以引用此笔记帮助回忆。
