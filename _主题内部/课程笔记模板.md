# `笔记/NN_标题.md` 模板

每个主题下有一个独立的 `笔记/` 文件夹，存放各门课的知识点深度笔记。用于记录**不是错题、不是跨主题兴趣、但值得深度记住的知识点**——洞见、关键公式、思维模型、"aha moment"。

文件名：`NN_标题.md`，课号与课程文件一一对应（如 `笔记/08_状态空间模型与动态系统入门.md` 对应 `课程/08_状态空间模型与动态系统入门.md`）。

模板：

```markdown
# [课程标题] 知识点笔记

## [知识点A]
- **核心洞见**：（用一句话说清本质）
- **为什么重要**：（这个知识点改变了你对什么的理解？）
- **类比/直觉**：（不用公式，怎么跟一个外行解释？）
- **关键公式/定义**：（如果有）

## [知识点B]
- **核心洞见**：
- **为什么重要**：
- **类比/直觉**：

---

🏷️ **关键词**：#关键词1 #关键词2 #关键词3

🔗 **关联**：
- ← 来自：[[../课程/NN_课程标题]]
- → 相关遗漏：[[../错题/NN]]（如果有）
- → 相关关联：[[../关联#知识点名]]（如果有）
```

> **使用规则**：
> - 这是一个**可选文件**——不是每门课都需要。只有遇到需要深度记录的知识点时才创建。
> - 知识点和错题的区别：错题是你**理解错了**的东西，知识点是你**理解了但想记住深度**的东西。
> - 关键词用 `#tag` 格式（Obsidian 兼容），方便全局搜索和知识图谱可视化。
> - 创建笔记时，同步在对应 `课程/NN_标题.md` 中插入反向链接：`> 📝 深度笔记：[[../笔记/NN_标题#知识点A]]`。
