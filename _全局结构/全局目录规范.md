# 全局目录规范

所有学习数据存放在根目录 `my-learning-log/` 下。

```
my-learning-log/
├── _全局索引.md                 # 所有主题的导航、进度摘要
├── _知识关联图谱.md             # 跨主题索引（仅摘要，详细关联下沉到各主题 关联.md）
├── _兴趣挖掘日志/               # 历史兴趣挖掘记录按月归档
│   ├── 股市交易兴趣生长记录.md  # 内容总结式命名
│   └── ...
├── [主题A]/
│   ├── 进度.md
│   ├── 课程规划.md                # 可选：未来课程大纲（标题+目标，非正文）
│   ├── 复习计划.md
│   ├── 关联.md                  # 本主题与其他主题的详细关联
│   ├── 课程/
│   │   ├── 01_核心概念.md
│   │   └── ...
│   ├── 笔记/                     # 可选：知识点深度笔记
│   │   └── ...
│   ├── 错题/
│   │   ├── _活跃遗漏.md         # 汇总当前所有未解决遗漏
│   │   ├── 01.md               # 第01课错题/遗漏详情
│   │   └── ...
│   └── 归档/
│       └── 复习记录_2026-07.md  # 旧复习记录按月或按需归档
├── [主题B]/
│   ├── 进度.md                  # 状态可标记为"探索中"（轻量结构，见探索中主题规范）
│   ├── 灵感笔记.md              # （仅探索中主题）链接、想法、随手记录
│   ├── 关联.md
│   └── ...
└── ...
```

主题内部结构详见 `_主题内部/主题结构说明.md`。
