# `_知识关联图谱.md` 模板

- 作用：**仅承担索引与导航职责**——主题级别的入口和一句极简桥接摘要，不展开详细解释。详细关联下沉到各主题内部的 `关联.md`。
- 健康阈值：当「跨主题索引」超过 **15 条**或「兴趣挖掘记录」超过 **20 条**时，助手主动提示用户进行一次手动整理（下沉/归档）。
- 支持 `[[主题名/路径]]` 形式的简单双向链接，兼容 Obsidian 等笔记工具。
- 结构模板：

```markdown
# 知识关联图谱

## 跨主题索引
- [[Python 基础]] ↔ [[统计学入门]]：列表推导式与数据清洗
- [[Python 基础]] ↔ [[分形与生成艺术]]：递归与自相似图形
- [[统计学入门]] ↔ [[机器学习偏差-方差]]：过拟合的统计直觉
→ 详见各主题内部 `关联.md`

## 兴趣挖掘记录（最近 30 天）
- 2026-07-17：学习 [[Python 基础/课程/05_递归]] 时，对分形图形表现出强烈兴趣。
  触发概念：递归、分形、科赫雪花
  当前关联课程：[[Python 基础/课程/05_递归]]
  → 已创建探索主题：[[分形与生成艺术]]

## 意向主题池
- 编译器原理（触发自 Python 解释器机制，已暂停探索，见 `_兴趣挖掘日志/编程语言学习兴趣记录.md`）

## 探索中的主题
- [[分形与生成艺术]]（探索中，详见其 `关联.md`）
```

> **兴趣挖掘记录归档规则**：在每次更新 `_知识关联图谱.md` 时，如果「兴趣挖掘记录」中存在超过 **30 天**的条目，立即将其移至 `_兴趣挖掘日志/` 中对应主题的归档文件。文件名采用内容总结式命名（如 `股市交易兴趣生长记录.md`），顶部保留"种子溯源"索引，标注每个兴趣最终的去向（已开课 / 放弃 / 暂停）。
