# 复习模式

## 0. 状态回顾

在开始复习前，先做简短的状态回顾：

> 欢迎回来！「[主题名]」当前有 N 个活跃遗漏点，M 门课复习到期（其中 X 门已过期）。

然后进入正式复习流程：

1. 读取 `进度.md`、`复习计划.md` 和 `错题/_活跃遗漏.md`。
2. 计算实际已过天数，确定到期项。
3. 按以下开场白开始复习：

> 你已经 N 天没有学习/复习「X」了。
> 当前到期复习：
> - 课程/01_核心概念：上次掌握 YYYY-MM-DD，下次复习 YYYY-MM-DD
> 重点检查遗漏：1. xxx；2. xxx
> 我们先不重读原文，直接做主动回忆。

复习时同样使用苏格拉底式提问。复习后的掌握判定规则与学习模式一致（见 `_规则/掌握程度判定.md`），但需要增加**复习失败时的间隔重置**：

- 完全掌握 → 复习次 +1，按间隔表计算下次复习日期。
- 基本掌握 → 复习次 +1，标记遗漏，下次复习优先检查。
- 部分理解/尚未理解 → **复习次重置为 1**，下次复习定为明天，遗漏写入 `错题/` 和 `_活跃遗漏.md`。

**所有复习日期的计算必须以实际完成复习的日期为起点，而不是原计划日期。**
