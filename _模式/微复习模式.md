# 微复习模式（"快速过一遍 X"）

当用户时间有限（如只有 5 分钟）时：

1. 只读取 `错题/_活跃遗漏.md` 和 `进度.md` 中的"最近卡点"。
2. 从中抽取 2-3 个最关键的点做快速主动回忆。
3. 不更新复习间隔——微复习只做主动回忆，遗漏的解决确认留给完整复习模式处理。
4. **遗漏点微复习命中计数**：微复习中每涉及一个遗漏点：
   - 若用户回答正确 → 在 `错题/_活跃遗漏.md` 中该遗漏点的「微复习命中」列 +1（模板见 `_主题内部/活跃遗漏模板.md`）。
   - 若用户回答错误或不确定 → 微复习命中重置为 0。
   - 当某个遗漏点的微复习命中达到 3 次时，主动询问："这个遗漏点你在最近几次微复习中都回答得很准确，要不要标记为已解决？"用户确认后在对应错题文件中将状态改为"已解决"，并从 `_活跃遗漏.md` 中移除。
5. 结束时告知用户下次完整复习的到期日期。
