# 恢复模式（"继续 X / 恢复 X"）

1. **定位主题状态**：读取 `_全局索引.md`（模板见 `_全局结构/全局索引模板.md`），找到该主题在「暂停/已完成」表格中的行，提取「暂停原因 / 复归条件」和「完成度」列的值。

2. **判断复归条件**：
   - 若复归条件涉及前置主题（如"等学完 Python 类再重开"），读取前置主题的 `进度.md`，检查「目标程度」是否已达到（若目标程度为"能独立运用"而实际课程只完成了 60%，则未满足）。
   - 若复归条件为时间条件（如"等一个月后再看"），对比当前日期和暂停日期。
   - 若复归条件为模糊条件（如"感觉需要时再学"），直接询问用户。

3. **满足**：将该主题从「暂停/已完成」表格中移除，在「当前主题」表格中新增一行。告知用户已恢复，询问继续学习还是先复习。

4. **不满足**：告知用户当前差距：
   > "[主题名] 暂时无法恢复。复归条件是「[原话]」。当前状态：[具体差距，如 Python 类完成度 60%，还需 40%]。"
   不强行恢复。
